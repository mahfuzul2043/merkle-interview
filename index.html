Instructions
Use arrow keys on your keyboard to control the cow. Or, at any time, you may provide a set of instructions for the cow to move. Use 'f' for forward, 'b' for backward, 'l' for left, and 'r' for right. A sample instruction string may look like this: 
ffbblr
(forward, forward, backward, backward, left, right). If your instruction string contains invalid inputs or invalid instructions, it will be disregarded, and only the valid instructions will be executed. You may instruct the cow to move with your keyboard simultaneously with an instruction string.
